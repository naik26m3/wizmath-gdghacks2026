Welcome, apprentice. Move the Rise and Run sliders to change the slope, or drag the y-intercept to slide the line up and down.

You

What is rise over run?

Oracle

Rise is how far the line travels vertically; run is how far it travels horizontally. Slope = rise ÷ run — a measure of how steeply the line climbs or falls.

You

And the y-intercept again?

Oracle

It's where the line crosses the y-axis — the value of y when x is 0. In y = mx + b, that's b.
